# Shadow Sports Trader
a sports analyzer of the current season. So far only supports NFL and NBA. Good way to analyze performance of players against the oppenent. It shows all active sports markets on Polymarket and then uses ESPNs hidden api to be able to track stats. It is a cool application meant to help look deeper into analytics.
### Frontend
react + ts

### Server:
python

install requirements.txt

run server using: `uvicorn main:ap --port <YOUR_PORT> --reload`
